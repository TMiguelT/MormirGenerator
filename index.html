Mormir Basic Generator

Most recent set: Battle for Zendikar

How to Use

  • Make a deck of 60 basic lands
  • Start a Cockatrice game as usual with this deck
  • Press Ctrl+T to open the tokens menu
  • Paste in Momir Vig, Simic Visionary Avatar to create a Mormir Avatar
  • Each time you activate Mormir's ability:
    • Set the relevant CMC below
    • Press the `Generate` button, and then `Copy Last` to add it to the clipboard
    • Open the tokens menu in Cockatrice (Ctrl+T)
    • Paste in the copied card name (Ctrl+V) into the name field without changing any other settings
