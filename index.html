MoJhoSto Generator

Most recent set: Oath of the Gatewatch

How to Use

  • Make a deck of 60 basic lands along with any of the 3 avatars you intend to use
  • Start a Cockatrice game as usual with this deck
  • Each time you activate an avatar's ability:
    • Set the relevant CMC below
    • Press the relevant `Generate` button, and then `Copy Last` to add it to the clipboard (this doesn't work for Jhoria since she generates 3 and you only pick one)
    • Open the tokens menu in Cockatrice (Ctrl+T)
    • Paste in the copied card name (Ctrl+V) into the name field without changing any other settings
